Exercise 1.1

Your sketch:

Instructions

Your task is to write a Star class that you will then use to create falling and shooting star animations.

Open sketch.js. You will see that it contains a function, star(), that draws a star at x and y coordinates provided as function arguments. You may recognise this function from an earlier practical. It is provided to help you implement the Star class. It will not be used in the finished sketch.

At the top of the file, create a class called Star. It should have two instance variables, x and y, that are initialised with values passed to the constructor. For example, to create a star at 100, 300, you would type:

let myStar = new Star(100, 300);

Next, add a method called draw(). This method should have no parameters. It should draw the star at the object's x and y coordinates. So, to draw the example star created above, you could type:

myStar.draw();

...and a star would appear at 100, 300.

Next, add two more methods, one called fall() and one called shoot(). Both methods should take one parameter called speed, a number representing the number of pixels the star should fall or shoot by respectively.

Implement the methods. fall() should move the star object along the y axis and shoot() should move the star object along the x axis. Here are examples of how the methods would be called on the example star object:

myStar.fall(2); // moves the star two pixels down the y axis
myStar.shoot(5); // moves the star five pixels along the x axis.

Outside the class, create an animation to test your methods. For example, the animation shown in the video below was created as follows (Star class not shown):

let fallingStar = new Star(200, 0);
let shootingStar = new Star(0, 200);
                
function setup() {
  createCanvas(400, 400);
}
                
function draw() {
  background(0);
  shootingStar.shoot(5);
  shootingStar.draw();
  fallingStar.fall(2);
  fallingStar.draw();
}

Check your work by running check-exercise1-1.html.

Expected output:

You will find it helpful to have the bouncingBallsOO example from lecture open to use as reference. You can find all example code in the PDM1-example-code repo.

The main thing to remember (that people often forget), is that when referencing instance variables from within the class, you need to type this. before the variable name. You can see lots of examples of this in the bouncingBallsOO example code.

Creating a method is exactly like creating a function, it just needs to be inside the class definition--that means completely between the opening and closing curly braces. You also should leave out the keyword function. There are examples in the bouncingBalls00 sketch.

To implement the draw() method, copy and paste the contents of the star() function into your draw() method (not the draw() function) and replace the uses of the x and y arguments in the original to use the the instance variables instead.

Implementing the fall() and move() methods should be straightforward. We didn't go over an example of a method with a parameter in class but the syntax is exactly the same as parameters in functions and the class constructor.
